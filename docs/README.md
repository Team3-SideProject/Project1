## 구조 개편안

- 문서 구조 설계안: `./docs-structure.md`

## 구조

- `requirements/`: 페이지 스펙, 기능 명세, 사용자 흐름
- `design/`: 디자인 시스템, 컬러 가이드, 와이어프레임
- `api/`: API 명세와 엔드포인트 목록
- `database/`: DB 테이블 명세, 스키마, ERD
- `architecture/`: 시스템 구조와 시퀀스 다이어그램
- `agents/`: AI 작업 규칙과 프론트/백엔드 에이전트 가이드
- `adr/`: 주요 기술 결정 기록
- `til/`: 기능 구현 후 학습 내용 정리 템플릿
- `meeting/`: 회의록
