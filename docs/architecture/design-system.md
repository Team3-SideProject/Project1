# 디자인 시스템 초안

## 방향성

- 어두운 트레이딩 터미널 느낌
- 고대비 텍스트와 명확한 수익/손실 색상
- 카드보다는 밀도 있는 패널과 테이블 중심 구성
- MVP에서는 복잡한 애니메이션보다 정보 가독성 우선

## 컴포넌트

| 컴포넌트 | 용도 |
| --- | --- |
| `Button` | 주요 액션, 보조 액션, 위험 액션 |
| `Input` | 로그인, 회원가입, 주문 수량 입력 |
| `Table` | 주식 목록, 포트폴리오, 거래 내역, 랭킹 |
| `Panel` | 대시보드 정보 묶음 |
| `Badge` | 상승/하락, 거래 유형 표시 |
| `Chart` | 가격 히스토리 표시 |

## 버튼 규칙

- 매수: 초록색 계열
- 매도: 빨간색 계열
- 일반 액션: 어두운 배경 + 밝은 텍스트
- 비활성 상태: 낮은 대비로 표시

## 텍스트 규칙

- 금액은 천 단위 콤마를 표시합니다.
- 상승/수익은 `+` 기호를 붙입니다.
- 하락/손실은 `-` 기호를 유지합니다.
