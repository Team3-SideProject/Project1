# 와이어프레임 초안

`wireframe.png`는 실제 화면 디자인 확정 후 추가합니다.  
초기 협업 단계에서는 아래 텍스트 와이어프레임을 기준으로 작업합니다.

## 로그인

```text
┌───────────────────────────┬─────────────────────┐
│ StockSim 소개/미니 차트    │ 로그인 폼            │
│                           │ 이메일              │
│                           │ 비밀번호            │
│                           │ 로그인 버튼          │
│                           │ 회원가입 버튼        │
└───────────────────────────┴─────────────────────┘
```

## 대시보드

```text
┌──────────┬──────────────────────────────────────┐
│ Sidebar  │ Topbar                               │
│          ├──────────┬──────────┬───────────────┤
│          │ 현금     │ 평가금   │ 손익          │
│          ├──────────┴──────────┴───────────────┤
│          │ 주식 목록 │ 차트        │ 주문 패널   │
│          ├──────────┴─────────────┴────────────┤
│          │ 포트폴리오 │ 거래 내역 / 랭킹        │
└──────────┴──────────────────────────────────────┘
```
