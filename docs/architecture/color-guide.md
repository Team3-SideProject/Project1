# 컬러 가이드

## 기본 팔레트

| 토큰 | 색상 | 용도 |
| --- | --- | --- |
| `background` | `#050706` | 전체 배경 |
| `surface` | `#0d100f` | 패널 배경 |
| `surface-muted` | `#151a18` | 보조 영역 |
| `border` | `rgba(255, 255, 255, 0.08)` | 경계선 |
| `text-primary` | `#eef5ef` | 주요 텍스트 |
| `text-secondary` | `#a4ada8` | 보조 텍스트 |
| `positive` | `#3df25d` | 상승, 수익, 매수 |
| `negative` | `#ff8a7a` | 하락, 손실, 매도 |
| `accent` | `#6fee83` | 강조 |

## 사용 규칙

- 수익/상승 정보는 초록색 계열만 사용합니다.
- 손실/하락 정보는 빨간색 계열만 사용합니다.
- 배경은 어두운 톤을 유지하되, 패널 간 경계는 얇은 border로 구분합니다.
