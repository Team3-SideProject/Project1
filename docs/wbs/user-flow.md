# 사용자 흐름

## 1. 회원가입/로그인

1. 사용자가 회원가입 페이지에 진입합니다.
2. 이메일, 닉네임, 비밀번호를 입력합니다.
3. 가입 성공 시 초기 가상 현금 10,000,000원이 지급됩니다.
4. 로그인 페이지에서 이메일과 비밀번호를 입력합니다.
5. 로그인 성공 시 JWT를 저장하고 메인 페이지로 이동합니다.

## 2. 주식 조회

1. 사용자가 메인 페이지에 진입합니다.
2. 서버에서 더미 주식 목록을 조회합니다.
3. 사용자가 종목을 선택합니다.
4. 주식 상세 페이지에서 현재가, 설명, 가격 히스토리를 확인합니다.

## 3. 매수

1. 사용자가 주식 상세 페이지에서 매수 수량을 입력합니다.
2. 화면에서 예상 매수 금액을 표시합니다.
3. 사용자가 매수 버튼을 클릭합니다.
4. 서버는 보유 현금이 충분한지 검증합니다.
5. 성공 시 현금, 포트폴리오, 거래 내역이 갱신됩니다.

## 4. 매도

1. 사용자가 보유 종목의 매도 수량을 입력합니다.
2. 화면에서 예상 매도 금액을 표시합니다.
3. 사용자가 매도 버튼을 클릭합니다.
4. 서버는 보유 수량이 충분한지 검증합니다.
5. 성공 시 현금, 포트폴리오, 거래 내역이 갱신됩니다.

## 5. 포트폴리오/랭킹

1. 사용자가 포트폴리오 페이지에서 총 자산과 보유 종목을 확인합니다.
2. 사용자가 거래 내역 페이지에서 매수/매도 기록을 확인합니다.
3. 사용자가 랭킹 페이지에서 총 자산 기준 상위 유저를 확인합니다.
